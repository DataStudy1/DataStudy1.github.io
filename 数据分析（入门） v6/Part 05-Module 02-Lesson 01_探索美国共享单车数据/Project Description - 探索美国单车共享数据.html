探索美国单车共享数据

在该项目中，你将编写 Python 代码来导入美国共享单车的数据，并通过描述性统计的计算来回答围绕它的有趣问题。你还将编写一个脚本，通过原始输入，在终端中创建交互式体验以呈现这类统计信息。

在你提交项目之前

检查审阅标准

你的项目将会由优达学城的审阅专家根据 项目审阅标准 进行评估。在你提交项目之前，请使用[ 项目审阅标准 ]来检查你的项目。如果你的提交已达到所有要求，则可以提交项目。如果你发现项目还有改进的空间，请继续改进吧，直到你的项目符合审阅标准中列明的所有要求！请特别要留意，项目审阅标准中包含对代码质量的要求。在该项目中，仅仅获得正确的答案并不是最重要的，具备良好的编码习惯来得到你的解决方案也很重要。

收集提交的材料

在该项目中，你需要提交如下两个文件：

  • bikeshare.py : 你的代码
  • readme.txt : 如果你在该项目中引用或参考了来自其他网站、书籍、资源，请一一在该文件中列明出处。

你不需要在提交项目时包含任何数据文件。

提交项目

当你准备好，请点击“提交项目”按钮，进入项目提交页面。你可以将你的文件（前述 bikeshare.py 和 readme.txt）作为 **a.zip **压缩文件提交。或者你可以链接到包含你项目文件的GitHub仓库。如果你使用GitHub，请注意，提交将是提交时所链接仓库的快照。建议你将每个项目保存在一个单独的仓库中，以避免任何潜在的混淆 ：如果审阅专家收到含有多个项目的文件夹，可能会对将要评估的项目有所混淆。

我们最多需要一周的时间来给你提交的项目评分，但在大多数情况下，评估时间要快得多。一旦你提交了邮件，你就会收到一封确认邮件。如果你对提交项目或希望检查你提交的状态有任何疑问，请在发邮件至 support@youdaxue.com。与此同时，你可以随时继续学习该课程的其他部分！
